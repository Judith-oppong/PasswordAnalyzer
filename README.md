# Password Analyzer

A real-time password strength evaluation tool built with Python and Streamlit.

## Features
- Hybrid evaluation: rule-based checks, entropy analysis, and leaked password detection
- Real-time colour-coded feedback and strength scoring
- Brute-force cracking time estimation
- Secure password and passphrase generation
- Session-based encrypted password vault
- Anonymous CSV logging with statistics dashboard

## How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run: streamlit run app.py

## Project Context
Built as a dissertation project exploring hybrid password strength evaluation.
